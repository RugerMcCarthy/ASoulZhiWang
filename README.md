# ASoulZhiWang
一个ASOUL评论区小作文APP，可以查询小作文出处，也可以浏览小作文库
查重API来自: https://github.com/stream2000/ASoulCnki
小作文库来自: https://asoul.icu

## 技术栈和引用库 👍
* JetPack Compose - 声明式UI框架
* Hilt - 依赖注入
* Navigation - 导航
* Retrofit - 访问Restful API
* Paging3 - 列表数据加载库
